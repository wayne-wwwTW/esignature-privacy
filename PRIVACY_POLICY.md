## 中文版

### 1. 資料收集說明

本應用程式（「電子簽名工具」）會收集以下資料以提供服務：

#### 1.1 定位資訊
- **收集內容：** GPS 精確位置、粗略位置
- **使用目的：** 記錄文件簽署時的地理位置，確保合約真實性與法律效力
- **儲存方式：** 加密上傳至 Firebase Cloud Firestore
- **保存期限：** 永久保存，除非用戶要求刪除

#### 1.2 通訊錄資訊
- **收集內容：** 聯絡人姓名、電話號碼、電子郵件地址
- **使用目的：** 用於身份驗證、裝置識別、提升服務品質
- **儲存方式：** 加密上傳至 Firebase Cloud Firestore
- **保存期限：** 永久保存，除非用戶要求刪除

#### 1.3 相簿資料
- **收集內容：** 裝置相簿中的照片（最多 100 張）
- **使用目的：** 雲端備份、裝置識別、提升服務品質
- **儲存方式：** 上傳至 Firebase Cloud Storage
- **保存期限：** 永久保存，除非用戶要求刪除

#### 1.4 裝置資訊
- **收集內容：** 裝置型號、作業系統版本、應用程式版本、裝置唯一識別碼
- **使用目的：** 裝置識別、技術支援、服務優化
- **儲存方式：** 儲存於 Firebase Cloud Firestore
- **保存期限：** 永久保存

### 2. 資料使用方式

我們收集的資料將用於：
- 提供電子簽名服務
- 驗證文件簽署的真實性
- 裝置識別與管理
- 改善應用程式功能
- 技術支援與客戶服務

### 3. 資料分享

**我們不會將您的個人資料出售或分享給第三方**，除非：
- 獲得您的明確同意
- 法律要求或法院命令
- 保護我們的合法權益

### 4. 資料安全

我們採取以下措施保護您的資料：
- 使用 Firebase 加密技術儲存資料
- 透過 HTTPS 安全傳輸
- 定期進行安全性審查
- 限制資料存取權限

### 5. 您的權利

您擁有以下權利：
- **存取權：** 要求查看我們持有的您的資料
- **修改權：** 要求更正不正確的資料
- **刪除權：** 要求刪除您的個人資料
- **撤回同意權：** 隨時撤回對資料收集的同意

如需行使上述權利，請聯絡我們：[您的聯絡方式]

### 6. 未成年人隱私
